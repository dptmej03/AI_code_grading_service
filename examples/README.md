# 테스트 데이터 및 샘플 파일

이 폴더는 시스템을 빠르게 테스트할 수 있는 샘플 파일들을 포함합니다.

## 📁 파일 목록

### 1. `answer.ipynb` (정답 파일)
모범 답안이 포함된 Jupyter 노트북

**내용:**
- **문제 1**: 1부터 n까지의 합 계산
  - 함수: `sum_numbers(n)`
  - 출력: `55` (n=10일 때)
  - 특징: 수식을 이용한 효율적 구현 (O(1))

- **문제 2**: 리스트 중복 제거 및 정렬
  - 함수: `remove_duplicates(lst)`
  - 출력: `[1, 2, 3, 4, 5]`
  - 특징: set과 sorted 사용

---

### 2. `20210001_홍길동.ipynb` (학생 1)
우수한 답안 (거의 완벽함)

**특징:**
- 문제 1: 루프를 사용한 정확한 구현
- 문제 2: 명확한 변수명 사용
- 출력값: 모두 일치
- **예상 점수**: 약 95/100

---

### 3. `20210002_김영희.ipynb` (학생 2)
부분적으로 완성된 답안 (오류 있음)

**특징:**
- 문제 1: 논리 오류 (range(1, num) → num까지 포함 안 됨)
  - 출력값: 54 (정답: 55)
- 문제 2: 출력값은 맞으나 변수명이 좋지 않음 (s, x, arr, dup 등)
- **예상 점수**: 약 65/100

3. **채점 기준 수정**
   - `rubric.json`에서 문제 개수, 배점, 항목 수정
   - 각 항목의 최대 점수 조정

## ⚠️ 주의사항

- **파일명 형식**: `학번_이름.ipynb` (예: `20210001_홍길동.ipynb`)
  - 이 형식이 아니면 대시보드에서 학번 파싱이 제대로 되지 않을 수 있음

- **노트북 형식**: Jupyter Notebook 4.x 이상
  - `.ipynb` 파일은 유효한 JSON 형식이어야 함

- **마크다운 헤더**: 문제별 분리를 위해 필수
  ```markdown
  ## 문제 1
  ## 문제 2
  ```
  또는
  ```markdown
  # 문제 1
  # 문제 2
  ```

- **인코딩**: 모든 파일을 UTF-8로 저장
  - 한글이 포함되면 반드시 확인
